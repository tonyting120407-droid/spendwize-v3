# SpendWize

SpendWize is a lightweight personal finance web app for tracking monthly earnings, estimated taxes, expenses, and remaining spending money.

## Features

- Dashboard interface with monthly income, after-tax income, total spending, and money left to spend.
- Budget health progress meter and categorized spending breakdown.
- Editing interface for updating income, tax rate, and expense entries.
- Local browser storage so changes remain after refreshes.

## Run locally

Open `index.html` in a browser, or serve the folder with any static file server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
