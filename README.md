# SpendWize

SpendWize is a lightweight personal finance web app for tracking monthly earnings, estimated taxes, daily expenditures, expenses, transaction history, and remaining spending money.

## Features

- Dashboard interface with monthly income, after-tax income, total spending, and money left to spend.
- Daily expenditure form on the main dashboard that auto-sorts purchases into categories such as Food, Accessories, Transportation, Shopping, Health, and more.
- Spending category breakdown with an "Add tracked expense" shortcut for adding more tracked spending.
- Budget health progress meter and categorized spending breakdown.
- Editing interface for updating income, tax rate, and expense entries.
- Transaction history page for reviewing every tracked and auto-sorted expense.
- Local browser storage so changes remain after refreshes.

## Run locally

Open `index.html` in a browser, or serve the folder with any static file server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
