# SpendWize

SpendWize is a lightweight personal finance web app for tracking daily expenditures, monthly or yearly fixed payments, saving goals, daily budgets, transaction history, and money left to spend.

## Features

- Dashboard summary focused on total amount spent and money left to spend.
- Daily expenditure form that auto-sorts purchases into categories such as Food, Accessories, Transportation, Shopping, Health, and more.
- Fixed payments editor for recurring monthly or yearly expenses such as rent, electric fees, insurance, subscriptions, and loans.
- Goals section for setting a monthly saving goal and daily spending budget.
- Transaction history split into separate Daily Expenditures and Fixed Payments sections.
- Budget health progress meter that considers after-tax income, fixed payments, daily spending, and savings goals.
- Local browser storage so changes remain after refreshes.

## Run locally

Open `index.html` in a browser, or serve the folder with any static file server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
